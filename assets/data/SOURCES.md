# Data Sources

- Quran text: Tanzil Project Uthmani text, downloaded from https://tanzil.net/pub/download/index.php on 2026-06-08. Tanzil terms require the source to be clearly indicated and linked.
- Adhkar/Duas: Hisn al-Muslim JSON repository by rn0x, raw source preserved in assets/data/hisn_almuslim_raw.json.
- Asma Ul Husna: Standard 99 Names list prepared from public reference tables; review by a qualified source is recommended before publishing.
